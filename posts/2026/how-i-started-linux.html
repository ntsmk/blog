Here is how I started using Linux and why I love it

I have installed Linux Mint Xfce on my old Macbook Air beginning of February, 2026. It was my first time to install different OS on my computer. Also first time to install Linux too. It has been about 3 weeks now. And I fell in love with this OS.

How I started it

One day, my friend said: "Linux is faster than Windows. You should try." His word was stuck with me. Before that, I have never used linux on my personal computer. Although, in the past, I have ran Linux on local VM, AWS EC2, and Raspberry Pi as a learning process.

Even so, I belive the experience between those and bare mental is completely different.

And shortly after, out of curiosity, I started searching about Linux. I started seeing more articles basically suggesting to switch to Linux from Windows because how Microsoft nowadays operates. Saying Windows computers are full of bloat which makes the computer slower.

Probably the timing was also right. Microsoft ended Windows 10 support on 2025 October. That means you either go upgarade to Windows 11 or go swith to different OS, which is MacOS or Linux. But what if the computer is old enough not to be able to upgrade to Windows 11?

If you want to stick with Windows, you will need to go buy newer laptop or use specific tool to skip the requirement for Windows 11 from old laptop(Which I acually did on the 2016 Windows laptop)
